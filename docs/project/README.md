# Project Documentation

Estado actual, roadmap, alcance y métricas del proyecto velox-terminal.

## Documents

| File | Purpose | Read when |
|------|---------|-----------|
| `PROJECT_STATE.md` | Estado actual del proyecto, completado y en progreso | Starting a session, need context on what's done |
| `NEXT_ACTIONS.md` | Próximas acciones priorizadas por fase | Planning what to work on next |
| `ROADMAP.md` | Roadmap por fases (núcleo inicial → mantenimiento) | Understanding long-term direction |
| `PROJECT_SCOPE.md` | Alcance del proyecto, qué está incluído y qué no | Evaluating if a feature belongs |
| `ASSUMPTIONS.md` | Suposiciones explícitas del proyecto | Validating design decisions |
| `RISK_REGISTER.md` | Riesgos del proyecto con mitigaciones | Risk assessment and planning |

## Recommended Loading Order

1. `PROJECT_STATE.md` — current state
2. `ROADMAP.md` — where we're going
3. `NEXT_ACTIONS.md` — what to do next
4. Deeper docs only if needed (risks, assumptions, scope)
